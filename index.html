Girls Coding Community

Welcome To Girls Coding Community!

Girls Coding Community is a website that provides resources of different coding languages for girls who want to know more about programming. Our website allows individuals to learn about HTML, CSS, and Python by the form of quizzes. Instead of purely reading information about coding languages, GCC provides an interactive way of developing knowledge of their selected coding language. Using this method not only teaches information, but it also allows individuals to become more intuitive. Being intuitive can help us make better decisions along with allowing us to gain more confidence in the decisions we make. Additionally, individuals can also create their own projects and engage in a community where they could receive feedbacks for their projects.
